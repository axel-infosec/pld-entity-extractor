# pld-entity-extractor

Herramienta automatizada para la extracción y clasificación de entidades (Personas) en textos no estructurados.

## Objetivo del Proyecto

Prototipo de automatización diseñado para la detección y extracción de entidades (Personas) en grandes volúmenes de texto no estructurado. Esta herramienta busca reducir la carga operativa en los procesos de Prevención de Lavado de Dinero (PLD), minimizando el error humano en la identificación de sujetos.

## Funcionalidades
- Procesamiento de Lenguaje Natural (NLP): Utiliza modelos pre-entrenados para la identificación precisa de entidades nominales (PER).

- Normalización de Datos: Limpieza automática de duplicados y ordenamiento alfabético para facilitar la gestión de listas.

- Reporteo Estructurado: Exportación automática de resultados a formato .xlsx para su integración con plataformas de análisis de datos.

## Tecnologías Utilizadas
- Python: Lenguaje principal de automatización.

- spaCy: Motor de IA para el reconocimiento de entidades.

- Pandas: Gestión de estructuras de datos y generación de reportes.
